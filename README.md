# CodSoft Java Internship Projects 🚀

This repository contains the projects completed during my **Java Programming Internship at CodSoft**.
Each task focuses on improving core **Java programming, logic building, Object-Oriented Programming (OOP), and problem-solving skills**.

---

## 📌 Task 1: Number Guessing Game

A simple Java program where the system generates a random number and the user tries to guess it.

### Features

* Generates a random number
* User guesses the number
* Provides hints like **Too High** or **Too Low**
* Displays number of attempts

### Concepts Used

* Java Random Class
* Loops
* Conditional Statements
* User Input Handling

---

## 📌 Task 2: Student Grade Calculator

A Java program that calculates **total marks, average percentage, and grade** based on marks entered for multiple subjects.

### Features

* Accepts marks for multiple subjects
* Calculates **Total Marks**
* Calculates **Average Percentage**
* Displays **Grade (A–F)** based on performance

### Concepts Used

* Loops
* Conditional Statements
* Arithmetic Operations
* User Input

---

## 📌 Task 3: ATM Interface

A menu-driven **ATM simulation system** that allows users to perform basic banking operations.

### Features

* Deposit Money
* Withdraw Money
* Check Account Balance
* Menu-driven interface

### Concepts Used

* Object-Oriented Programming (OOP)
* Classes and Methods

## 📌 Task 4: Currency Converter

### Concepts Used

## 📌 Task 5: Student Management System

A Java-based system to manage student records with file storage.

### Features

### Concepts Used
